1. Copy the whole folder myPhonegapApp to your Desktop or wherever is convenient for you.
2. Open up a terminal/command prompt and navigate to the myApp folder.
3. Type "phonegap serve" (need to download latest version of phonegap desktop).
   Take note of the IP Address and port phonegap is listening on.
4. On your phone make sure you have a wifi connection.
5. Download the phonegap developer app on your phone if you haven't got it already.
6. Run the phonegap developer app and enter in the IP Address and port that phonegap
   is listening on.
7. Now that you're connected, any changes you make to the files inside of the
   myPhonegapApp/www folder will be immediately reflected in the phonegap developer app.
